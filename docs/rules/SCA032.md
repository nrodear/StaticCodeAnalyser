# SCA032 — Circular DataSource / Master-Detail loop

## Description

A cycle in the master-detail graph causes infinite recursion during BeforeOpen or any refresh and stack-overflows the process. Break the cycle by removing one of the links.

## Examples

### Bad (triggers the rule)

```pascal
QA.MasterSource = dsB   QB.MasterSource = dsA   // cycle
```

### Good (idiomatic fix)

```pascal
(make one query the parent, the other the detail)
```

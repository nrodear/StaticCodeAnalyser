# SCA015 — String literal repeated across multiple sites

![Severity](https://img.shields.io/badge/severity-Hint-blue) ![Type](https://img.shields.io/badge/type-Code%20Duplication-blueviolet)

> Same string literal appears N+ times - extract to constant

## Description

Repeated strings are change-coupling hazards (typo in one place silently diverges from the others). Extract to a const, especially for user-facing messages.

## Examples

### Bad (triggers the rule)

```pascal
ShowMessage('File not found');
... 
LogError('File not found');
```

### Good (idiomatic fix)

```pascal
const MSG_FILE_NOT_FOUND = 'File not found';
ShowMessage(MSG_FILE_NOT_FOUND);
```

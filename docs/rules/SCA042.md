# SCA042 — God event handler

![Severity](https://img.shields.io/badge/severity-Hint-blue) ![Type](https://img.shields.io/badge/type-Code%20Smell-yellow)

> Single method wired to >= N component events (default N=5)

## Description

Spaghetti indicator: one handler dispatching dozens of events is hard to read, hard to change, and almost always has cohesion problems. Split by responsibility.

## Examples

### Bad (triggers the rule)

```pascal
btn1.OnClick = HandleEverything
btn2.OnClick = HandleEverything
btn3.OnClick = HandleEverything
btn4.OnClick = HandleEverything
btn5.OnClick = HandleEverything
```

### Good (idiomatic fix)

```pascal
(one handler per logical action)
```

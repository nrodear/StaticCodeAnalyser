# SCA049 — Length(s) - N without guard

![Severity](https://img.shields.io/badge/severity-Hint-blue) ![Type](https://img.shields.io/badge/type-Bug-red)

> Length / .Count with subtraction - native-uint underflow when empty

## Metadata

## Description

`Length(s) - 1` on an empty string evaluates to 0 - 1 = MaxUInt under NativeUInt arithmetic and indexes into garbage. Guard for emptiness or cast to NativeInt.

## Examples

### Bad (triggers the rule)

```pascal
Last := s[Length(s) - 1];
```

### Good (idiomatic fix)

```pascal
if Length(s) >= 2 then Last := s[Length(s) - 1];
```

# SCA020 — Empty method body

![Severity](https://img.shields.io/badge/severity-Hint-blue) ![Type](https://img.shields.io/badge/type-Code%20Smell-yellow)

> Method body has no statements

## Description

Empty method may indicate a forgotten implementation, a TODO that was never followed up, or an interface stub. Make intent explicit (assert, exception, or comment).

## Examples

### Bad (triggers the rule)

```pascal
procedure DoStuff;
begin
end;
```

### Good (idiomatic fix)

```pascal
procedure DoStuff;
begin
  raise ENotImplemented.Create('...');
end;
```

# SCA018 — Block nesting exceeds threshold

![Severity](https://img.shields.io/badge/severity-Hint-blue) ![Type](https://img.shields.io/badge/type-Code%20Smell-yellow)

> Nested if/for/while depth higher than configured maximum (default 4)

## Description

Deep nesting hurts readability and indicates the method is doing too much. Use guard clauses (early Exit) or extract inner blocks into helper methods.

## Examples

### Bad (triggers the rule)

```pascal
if a then
  if b then
    if c then
      if d then DoStuff;
```

### Good (idiomatic fix)

```pascal
if not a then Exit;
if not b then Exit;
if not c then Exit;
if d then DoStuff;
```

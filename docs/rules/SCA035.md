# SCA035 — Required field only on hidden controls

![Severity](https://img.shields.io/badge/severity-Warning-orange) ![Type](https://img.shields.io/badge/type-Bug-red)

> TField with Required=True is bound only to Visible=False controls

## Metadata

| Field | Value |
|---|---|
| **Rule ID** | `SCA035` |
| **Kind** | `DfmRequiredFieldNotVisible` |
| **Default severity** | Warning |
| **Type** | Bug |
| **Detector unit** | `uDfmRequiredField.pas` |
| **Tags** | `dfm`, `ux`, `required-field` |

## Description

Control exists but the user cannot see or interact with it - inserts fail every time. Make at least one bound control visible or drop the Required=True.

## Examples

### Bad (triggers the rule)

```pascal
DBEdit1.DataField = 'Email'   DBEdit1.Visible = False
```

### Good (idiomatic fix)

```pascal
(make the control visible or remove Required=True)
```

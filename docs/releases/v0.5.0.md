## v0.5.0 — Pre-release

First public version of the **Static Code Analysis Tool for Delphi**.
21 AST-based detectors, IDE plugin (RAD Studio 12 Athens) and
standalone VCL app.

### Features

- **AST-based analysis** with custom lexer + parser (uParser2),
  iterative traversal
- **Sonar-style classification**: Bug / CodeSmell / Vulnerability /
  SecurityHotspot / CodeDuplication
- **Quality score** (weighted sum, lower = better)
- **Sonar-style tile UI** in the IDE plugin (severity + type buckets)
- **Branch-Changes mode**: analyse only `.pas` files changed in
  Git/SVN
- **VCS integration** with timeout + process kill (60 s hard limit)
- **Claude AI prompt generation** per finding (Markdown with code
  context and before/after)
- **Suppression** via `// noinspection <Kind>` comments
- **Export**: HTML / JSON / CSV / Jira markup / plain text
- **DE localisation** of the UI (built-in dictionary)
- **Repo settings** via `analyser.ini` (BaseBranch,
  IncludeWorkingTree, Tortoise paths)
- **Ignore list** via `ignore.txt` (glob patterns)
- **Tortoise Git/SVN compatibility** (PATH search + standard
  install paths)

### Detectors

**Memory / Bugs:** MemoryLeak, FieldLeak, NilDeref, MissingFinally,
DivByZero, FormatMismatch
**Security:** SQLInjection, HardcodedSecret, HardcodedPath
**Code smells:** EmptyExcept, EmptyMethod, DeadCode, UnusedUses,
DebugOutput, LongMethod, LongParamList, DeepNesting, MagicNumber,
TodoComment
**Duplication:** DuplicateString, DuplicateBlock

### Requirements

- Windows 10/11
- (Plugin) RAD Studio 12 Athens
- (Standalone) none — compiles as a single EXE

### Known limitations

- No binaries in this release — source only. Build the plugin (BPL)
  and standalone (EXE) yourself.
- Floating-point division (`/`) is not checked by the DivByZero
  detector.
- Thresholds (LongMethod = 50 lines etc.) are currently hardcoded.
